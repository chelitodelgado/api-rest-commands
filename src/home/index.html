Introducción

Esta documentación te ayudara a familiarizarte con los recursos de la API de Linux Commands y te mostrará cómo realizar diferentes consultas, para que pueda sacarle el máximo partido.

REST

BASE: https://commands-linux.herokuapp.com/v1/commands

La URL base contiene información sobre todos los recursos de API disponibles. Todas las solicitudes son solicitudes GET y pasan por https. Todas las respuestas devolverán datos en json.
GET https://commands-linux.herokuapp.com/v1/commands

Actualmente la información disponible son:

  • command: Muestra todos los comandos disponibles.
  • description: Muestra una breve descripción del comando.

    • Información y paginación

    Esta API organiza automaticamenete la información en una sola pagina, por lo tanto no es necesrio realizar una paginación complicada. Cada documento consta de 27 categorias con n cantidad de comandos.

    Llave Tipo Descripción
    totaldocs int Total de documentos
    limit int Limite total de documentos actuales
    totalPages int Cantidad de paginas
    page int Pagina actual
